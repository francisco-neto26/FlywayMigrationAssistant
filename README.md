# Flyway Migration Assistant

Ferramenta desktop profissional para gestão de migrações SQL Flyway com interface gráfica moderna.

##  Funcionalidades

- ✅ Criação de migrações com nomenclatura padronizada.
- ✅ Sincronização automática de módulos
- ✅ Syntax highlighting SQL colorido
- ✅ Templates customizáveis
- ✅ Lembrança do último diretório usado
- ✅ Tema escuro moderno (FlatLaf Darcula)
- ✅ Gerenciamento de prefixos por módulo

## Tecnologias

- **Java 21**
- **Swing** + **FlatLaf** (UI moderna)
- **Maven** (build)
- **Google Guice** (Dependency Injection)
- **SLF4J + Logback** (Logging)
- **JUnit 5 + Mockito** (Testes)

### Pré-requisitos

	Eclipse Adoptium jdk-21.0.9.10-hotspot
